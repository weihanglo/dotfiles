⚙️ Dotfiles for macOS/Linux!!
====================================

Usage
-----
Just clone and run script!

```bash
git clone https://github.com/weihanglo/dotfiles.git ~/.dotfiles
. ~/.dotfiles/makeconfig.bash
```
